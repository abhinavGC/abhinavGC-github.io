ComboBox Control

ComboBox for WPF and Silverlight allows you to create customized, rich applications by using some of the features given below.

  • Auto-Searchable Drop-Down List - ComboBox automatically searches the list and selects items for you as you type the first few characters.

  • Populate the Drop-down List with Data Templates - ComboBox fully supports data templates, making it easy to add any visual elements to the list items. This includes text, images, and any other controls. The control uses element virtualization, so it always loads quickly, even when populated with hundreds of items.

  • Time-tested, Familiar Object Model - ComboBox has a rich object model. You can easily specify whether the end user is able to enter items that are not on the drop-down list, get or set the index of the selected item, the height of the drop-down list, and more.
